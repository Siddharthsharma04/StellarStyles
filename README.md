# StellarStyles 👗✨

**StellarStyles** is a premium e-commerce landing page designed for a modern clothing brand. It focuses on elegance, minimalism, and providing a seamless shopping experience for the fashion-forward individual.

## 🚀 Features
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Hero Section:** A high-impact introduction showcasing the brand's core philosophy: "Quality meets elegance."
- **Curated Collection:** A featured product grid showcasing the latest arrivals, including minimalist dresses, silk blouses, and linen suits.
- **Interactive UI:** Smooth navigation and a functional cart counter for enhanced user engagement.
- **Premium Aesthetics:** Styled with a sophisticated color palette and elegant typography (Montserrat & Playfair Display).

## 🛠️ Tech Stack
- **HTML5:** Semantic markup for structure and accessibility.
- **CSS3:** Custom layouts using Flexbox and CSS Grid.
- **JavaScript:** Dynamic interactivity for the shopping bag and navigation.
- **Google Fonts:** Integration of premium web fonts.
- **FontAwesome:** Scalable vector icons for a modern look.

## 📂 Project Structure
- `index.html`: The core landing page structure.
- `style.css`: Custom styling for the entire brand experience.
- `script.js`: Logic for interactive elements and cart functionality.

## 🏁 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Siddharthsharma04/StellarStyles.git
   ```
2. Open `index.html` in any modern web browser to view the site.
